# generate:form:alter
Generate an implementation of hook_form_alter() or hook_form_FORM_ID_alter

**प्रयोग:**
```
$ drupal generate:form:alter [options]
$ gfa  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | The Module name.
--form-id | Form ID to alter
--inputs | Create inputs in a form.
