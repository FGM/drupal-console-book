# theme:download
आवेदन में विषय डाउनलोड करे

**प्रयोग:**
```
$ drupal theme:download [arguments] [options]
$ td  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--composer | Use --composer option for manage the theme download with Composer

## उपलब्ध तर्क
तर्क | विवरण
---------|-------------
theme | the Theme name
version | Theme version i.e 1.x-dev
