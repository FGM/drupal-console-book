# translation:stats
Generate translate stats

**प्रयोग:**
```
$ drupal translation:stats [arguments] [options]
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--format | Define output format table|markdown

## उपलब्ध तर्क
तर्क | विवरण
---------|-------------
language | Language to generate translation stats against English
