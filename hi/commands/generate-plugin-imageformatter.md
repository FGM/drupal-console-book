# generate:plugin:imageformatter
Generate image formatter plugin.

**प्रयोग:**
```
$ drupal generate:plugin:imageformatter [options]
$ gpif  
```

## उपलब्ध विकल्प
विकल्प | विवरण
-------|-------------
--module | The Module name.
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
