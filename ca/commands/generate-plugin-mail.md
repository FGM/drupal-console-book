# generate:plugin:mail
Generate a plugin mail

**Ús:**
```
$ drupal generate:plugin:mail [options]
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--module | Nom del mòdul.
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
--services | Carregar serveis des del contenidor.
