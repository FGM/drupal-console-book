# rest:debug
Mostrar l'actual recurs Rest de l'aplicació

**Ús:**
```
$ drupal rest:debug [arguments] [options]
$ rede  
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--authorization | Rest resource status habilitat | deshabilitat

## Arguments disponibles
Argument | Detalls
---------|-------------
resource-id | Identificador Rest
