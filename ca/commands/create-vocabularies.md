# create:vocabularies
Crear vocabulari 'dummy' per l'aplicació Drupal 8.

**Ús:**
```
$ drupal create:vocabularies [options]
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--limit | Quants vocabularis voldrieu crear
--name-words | Màxim de paraules utilitzades al nom del vocabulari
