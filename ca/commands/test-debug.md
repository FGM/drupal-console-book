# test:debug
Llistar els Test Units disponibles per l'aplicació.

**Ús:**
```
$ drupal test:debug [arguments] [options]
$ td  
```

## Opcions disponibles
Opció | Detalls
-------|-------------
--test-class | Classe de verificació

## Arguments disponibles
Argument | Detalls
---------|-------------
group | Grup
