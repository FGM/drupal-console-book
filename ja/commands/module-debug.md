# module:debug
利用可能なモジュールを表示

**使い方:**
```
$ drupal module:debug [arguments] [options]
$ mod  
```

## 利用可能なオプション
オプション | 詳細
-------|-------------
--status | モジュールのステータス [enabled|disabled]
--type | モジュールのタイプ [core|no-core]

## 利用可能な引数
引数 | 詳細
---------|-------------
module | commands.module.debug.module
