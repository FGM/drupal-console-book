# module:download
モジュールをダウンロード

**使い方:**
```
$ drupal module:download [arguments] [options]
$ md  
```

## 利用可能なオプション
オプション | 詳細
-------|-------------
--path | プロジェクトのパス
--latest | 最新バージョンをダウンロードする
--composer | Composerを利用してモジュールをダウンロードする
--unstable | commands.module.install.options.unstable

## 利用可能な引数
引数 | 詳細
---------|-------------
module | モジュール名 (複数指定する場合はスペース区切り)
