# generate:entity:bundle
Generate a new content type (node / entity bundle)

**使い方:**
```
$ drupal generate:entity:bundle [options]
$ geb  
```

## 利用可能なオプション
オプション | 詳細
-------|-------------
--module | モジュール名
--bundle-name | The content type's machine name
--bundle-title | The content type's human-readable name
