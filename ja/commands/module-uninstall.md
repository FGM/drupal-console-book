# module:uninstall
モジュールをアンインストール

**使い方:**
```
$ drupal module:uninstall [arguments] [options]
$ mou  
```

## 利用可能なオプション
オプション | 詳細
-------|-------------
--force | 依存関係を無視してモジュールを強制的にアンインストールしますか？
--composer | Composerを利用してモジュールをアンインストールする

## 利用可能な引数
引数 | 詳細
---------|-------------
module | モジュール名を入力してください
