# module:install
モジュールをインストール

**使い方:**
```
$ drupal module:install [arguments] [options]
$ moi  
```

## 利用可能なオプション
オプション | 詳細
-------|-------------
--latest | 最新バージョンをダウンロードする
--composer | Composerを利用してモジュールをアンインストールする

## 利用可能な引数
引数 | 詳細
---------|-------------
module | 有効にするモジュール名 (複数指定する場合はスペース区切り)
