# generate:twig:extension
Generate a Twig extension.

**Usage:**
```
$ drupal generate:twig:extension [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--module | The Module name.
--name | Twig Extension name
--class | Class name
--services | Load services from the container.
