# theme:path
Returns the relative path to the theme (or absolute path)

**Usage:**
```
$ drupal theme:path [arguments] [options]
```

## Các tùy chọn có sẵn
Tùy chọn | Các chi tiết
-------|-------------
--absolute | Return theme absolute path

## Các đối số có sẵn
Đối số | Các chi tiết
---------|-------------
module | The Theme name (machine name)
