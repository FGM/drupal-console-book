# module:path
Returns the relative path to the module (or absolute path)

**Folosire:**
```
$ drupal module:path [arguments] [options]
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--absolute | Return module absolute path

## Argumente disponibile
Argument | Detalii
---------|-------------
module | The Module name (machine name)
