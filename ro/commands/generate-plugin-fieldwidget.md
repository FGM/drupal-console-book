# generate:plugin:fieldwidget
Generate field widget plugin.

**Folosire:**
```
$ drupal generate:plugin:fieldwidget [options]
$ gpfw  
```

## Opțiuni disponibile
Opțiune | Detalii
-------|-------------
--module | The Module name.
--class | Plugin class name
--label | Plugin label
--plugin-id | Plugin id
--field-type | Field type the plugin can be used with
