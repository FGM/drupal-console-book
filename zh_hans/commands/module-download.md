# module:download
下载模块

**用法:**
```
$ drupal module:download [arguments] [options]
$ md  
```

## 可用选项
选项 | 详细
-------|-------------
--path | 贡献项目路径
--latest | 默认下载最新版本
--composer | 使用 Composer 下载模块
--unstable | commands.module.install.options.unstable

## 可用参数
参数 | 详细
---------|-------------
module | 要启用的模块之间用空格隔开
