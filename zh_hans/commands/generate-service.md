# generate:service
Generate service

**用法:**
```
$ drupal generate:service [options]
$ gs  
```

## 可用选项
选项 | 详细
-------|-------------
--module | The Module name.
--name | commands.generate.service.options.name
--class | Class name
--interface | commands.common.service.options.interface
--services | Load services from the container.
--path_service | Path
